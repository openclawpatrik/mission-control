# Daily Memory Sync Spec

Goal: extract key points from today's chat and sync to durable memory files.

Targets:
- DECISIONS.md (important decisions)
- MEMORY.md (long-term signals)
- NOTES.md (short references/policies)
- KANBAN.json (actionable tasks)
- VOICE_INBOX.md (voice-derived items)

Rules:
- Keep entries concise.
- Avoid duplication.
- Preserve dates and source context.
